View Interface for nemo enhanced with nested locators and collection mechanics

Nemo Plugin to Use selenium-webdriver to take a screenshot

nemo-accessibility

nemo-accessibility plugin uses PAET api to run accessibility scans on a given page.

nemo-perfectomobile

PerfectoMobile plugin for NemoJS

nemo-wd-bridge

Nemo plugin to add wd driver capability to nemo

nemo-appium

Start an appium server during nemo startup, kill upon driver.quit

nemo-firefox-profile

Adds custom firefox preferences to nemo firefox driver instance

nemo-browsermob-proxy

BrowserMob Proxy plugin for for Nemo automation framework

nemo-saucelabs

nemo-window

Browser window extensions for Nemo automation

nemo-logger

selenium-webdriver logging initializer plugin for nemo

nemo-poster

plugin for posting a form via webdriver and nemo
